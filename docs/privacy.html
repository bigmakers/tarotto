プライバシーポリシー

Tarotto - タロット占い

最終更新日: 2026年2月22日

1. はじめに

Tarotto（以下「本アプリ」）は、ユーザーのプライバシーを最大限に尊重します。本プライバシーポリシーでは、本アプリにおけるデータの取り扱いについてご説明します。

2. 収集するデータ

本アプリは、ユーザーの個人情報を一切収集しません。

すべてのデータはユーザーのデバイス内にのみ保存され、外部サーバーに送信されることはありません。

データの種類 保存場所
ユーザー名 デバイス内（UserDefaults）
言語設定 デバイス内（UserDefaults）
占い履歴 デバイス内（SwiftData）
APIキー デバイス内（Keychain）
購入情報 Apple経由（App Store）

3. 外部サービスとの通信

本アプリは、以下の場合にのみ外部サーバーと通信します：

  • AI解釈機能の利用時 ユーザーが自分で設定したAPIキーを使用して、選択したAIプロバイダー（OpenAI、Anthropic Claude、Google Gemini）のサーバーに占い結果のテキストデータを送信します。
  • アプリ内課金の処理時 — AppleのApp Storeサーバーと通信します。

AI解釈機能は任意であり、APIキーを設定しない場合は外部通信は行われません。送信されるデータにはカードの配置と占いの質問内容のみが含まれ、ユーザーの個人情報は含まれません。

4. トラッキング

本アプリはユーザーのトラッキングを一切行いません。

  • 広告トラッキングなし
  • アナリティクスツールなし
  • サードパーティSDKなし（StoreKitを除く）
  • IDFA（広告識別子）の取得なし

5. データの保護

APIキーはAppleのKeychainに暗号化して保存されます。その他の設定データはデバイスのローカルストレージに保存され、デバイスのセキュリティ機能によって保護されます。

6. データの削除

ユーザーはいつでも設定画面の「全設定消去」機能を使用して、アプリ内に保存されたすべてのデータを削除できます。また、アプリをデバイスからアンインストールすることで、すべてのデータが完全に削除されます。

7. 子どものプライバシー

本アプリは、13歳未満のお子様を対象としたものではありません。13歳未満のお子様から意図的に個人情報を収集することはありません。

8. ポリシーの変更

本プライバシーポリシーは、必要に応じて更新されることがあります。重要な変更がある場合は、アプリ内でお知らせします。

9. お問い合わせ

プライバシーに関するご質問やご懸念がございましたら、以下までご連絡ください。

support@tarotto.app

Privacy Policy

Tarotto - Tarot Reading

Last updated: February 22, 2026

1. Introduction

Tarotto ("the App") respects your privacy. This Privacy Policy explains how we handle data within the App.

2. Data Collection

This App does not collect any personal information.

All data is stored exclusively on your device and is never transmitted to external servers.

Data Type Storage Location
User name On-device (UserDefaults)
Language setting On-device (UserDefaults)
Reading history On-device (SwiftData)
API keys On-device (Keychain)
Purchase info Via Apple (App Store)

3. External Communications

The App communicates with external servers only in the following cases:

  • When using AI interpretation — Using API keys that you configure, the App sends reading text data to your chosen AI provider (OpenAI, Anthropic Claude, or Google Gemini).
  • In-App Purchase processing — Communication with Apple's App Store servers.

The AI interpretation feature is optional. If you do not configure an API key, no external communication occurs. Only card placement and reading question data are transmitted; no personal information is included.

4. Tracking

This App does not track users in any way.

  • No advertising tracking
  • No analytics tools
  • No third-party SDKs (except StoreKit)
  • No IDFA collection

5. Data Security

API keys are encrypted and stored in Apple's Keychain. Other settings data is stored in the device's local storage and is protected by the device's security features.

6. Data Deletion

You can delete all data stored within the App at any time using the "Reset All Settings" feature in Settings. Uninstalling the App from your device will also completely remove all data.

7. Children's Privacy

This App is not intended for children under the age of 13. We do not knowingly collect personal information from children under 13.

8. Policy Changes

This Privacy Policy may be updated as needed. Significant changes will be communicated within the App.

9. Contact

If you have any questions or concerns about privacy, please contact us at:

support@tarotto.app
